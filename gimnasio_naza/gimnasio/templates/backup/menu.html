Guía y Especificaciones del Sistema

Requisitos Técnicos:
  • Instancia de MySQL Server activa y accesible.
  • El usuario configurado debe poseer privilegios globales de lectura y escritura (`GRANT ALL PRIVILEGES`).
  • Asegurar espacio en disco suficiente antes de restaurar volcados masivos.
Estructura de Archivos:
  • Nomenclatura automática: `backup_completo_YYYYMMDD_HHMMSS.sql`.
  • Formato plano estándar compatible con cualquier gestor (DBMS).
  • Mantiene íntegras las restricciones transaccionales y llaves foráneas (`FK`).
